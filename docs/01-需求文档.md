## 项目概述

**产品名称：** 别忘带了

**产品定位：** 一款面向日常出门场景的智能物品清单工具，帮助用户根据不同场景快速核对随身物品，防止遗漏。

**目标用户：** 经常在不同场合出门、容易忘带东西的人群。

**运行平台：** Mac + iOS（通过 PWA 实现，浏览器打开即可使用）

---

## 功能需求

### 1. 场景管理

| 功能 | 说明 |
|------|------|
| 预设场景 | 系统内置：看剧、桌游、聚餐、有氧、撸铁、散步、出差、回学校报道 |
| 新建场景 | 用户可以创建自定义场景，输入场景名称和可选图标 |
| 置顶场景 | 常用场景可置顶到列表最前面 |
| 删除场景 | 可删除不再需要的场景 |
| 使用频次统计 | 自动记录每个场景的使用次数和最近一次使用日期 |

### 2. 物品清单

| 功能 | 说明 |
|------|------|
| 物品添加 | 在每个场景下添加需要携带的物品 |
| 复选框勾选 | 每次出门前逐项打钩确认，勾完即为"成功出门" |
| 物品详情页 | 点击物品进入详情，包含照片、图标、存放位置 |
| 照片上传 | 用户可为物品上传照片（存于浏览器本地） |
| 图标选择 | 用户可选择一个图标（Emoji 或上传图片） |
| 存放位置 | 可记录物品的常用存放位置（如"玄关第二个抽屉"） |

### 3. 出门流程

| 功能 | 说明 |
|------|------|
| 成功出门 | 一个场景下所有物品勾选完成后，触发"成功出门" |
| 鼓励弹窗 | 成功出门后弹出鼓励文案，提醒用户享受生活 |
| 状态重置 | 下次再进入同一场景时，所有复选框恢复为未勾选状态 |

### 4. 搜索

| 功能 | 说明 |
|------|------|
| 全局搜索 | 可搜索场景名称和物品名称 |
| 即时筛选 | 输入关键词实时过滤显示结果 |

### 5. UI / UX

| 要求 | 说明 |
|------|------|
| 主色调 | 浅黄色 + 白色 |
| 设计风格 | 简洁、直观、一目了然 |
| 交互反馈 | 按钮点击、勾选等操作有即时视觉反馈 |
| 响应式 | 适配 Mac 浏览器和 iPhone 屏幕 |

---

## 数据存储

- 使用浏览器本地存储（LocalStorage / IndexedDB）
- 无需用户注册登录
- 数据完全存储在用户设备本地
- 支持导出备份（后续可考虑）

---

## 非功能需求

- 零服务器依赖（纯前端 PWA）
- 离线可用（Service Worker 缓存）
- 单页应用体验（SPA）
- 首屏加载快速
