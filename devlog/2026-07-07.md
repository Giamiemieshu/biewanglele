### DataStore API 一览
| 方法 | 用途 |
|------|------|
| `getScenes()` / `getScene(id)` | 获取场景列表/单个 |
| `addScene({name, icon})` | 新建场景 |
| `updateScene(id, updates)` | 更新场景 |
| `deleteScene(id)` | 删除场景（连带删除物品和记录） |
| `togglePinScene(id)` | 置顶/取消置顶 |
| `getSortedScenes()` | 置顶优先排序 |
| `getItems(sceneId)` / `getItem(id)` | 获取物品列表/单个 |
| `addItem({sceneId, name, icon})` | 添加物品 |
| `updateItem(id, updates)` | 更新物品（含照片、存放位置） |
| `deleteItem(id)` | 删除物品 |
| `addCheckoutRecord(sceneId, count)` | 记录一次成功出门 |
| `search(query)` | 搜索场景和物品 |
| `exportData()` / `importData(json)` | 数据备份/恢复 |
| `clearAll()` | 重置为初始数据 |

## 滚动问题修复

- [x] **问题**：物品详情页内容超长时被遮挡，无法滚动
- [x] **方案**：将 #app 改为 flex 全屏布局

  - `#app-header` 固定顶部（`flex-shrink: 0`）
