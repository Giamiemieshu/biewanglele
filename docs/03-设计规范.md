## 色彩体系

| 角色 | 色值 | 用途 |
|------|------|------|
| 主背景 | `#FFFDF5` | 页面整体背景（极浅米白） |
| 卡片背景 | `#FFFFFF` | 场景卡片、详情卡片 |
| 主色调 | `#F5D742` | 按钮、选中态、强调元素（暖黄） |
| 主色调深 | `#E8C635` | 按钮 hover |
| 文字主色 | `#2C2C2C` | 标题、主要文字 |
| 文字次要 | `#8C8C8C` | 辅助文字、时间戳 |
| 分割线 | `#F0EDE0` | 列表分割、边界 |
| 成功绿 | `#52C41A` | 已勾选、已完成 |
| 警示红 | `#FF4D4F` | 删除操作 |

## 字体

- 系统字体栈：
  ```
  -apple-system, BlinkMacSystemFont, "Segoe UI", 
  "PingFang SC", "Hiragino Sans GB", 
  "Microsoft YaHei", sans-serif
  ```
- 字号层级：14px（辅助）/ 16px（正文）/ 20px（卡片标题）/ 24px（页面标题）

---

## 间距系统

- 基础间距以 8px 为增量：8、16、24、32
- 卡片内边距：16px
- 卡片间距：12px
- 页面两侧边距：20px

---

## 圆角

- 卡片圆角：12px
- 按钮圆角：8px
- 输入框圆角：8px
- 头像/图标圆角：50%（圆形）

---

## 组件规范

### 场景卡片
```
┌────────────────────────┐
│  💪 撸铁             > │
│  已使用 12 次 · 上次: 7/5 │
└────────────────────────┘
```
- 左侧：图标（Emoji / 用户上传缩略图）
- 中间：场景名称 + 频次信息
- 右侧：箭头 / 置顶标识
- 点击进入详情

### 物品行
```
┌──────────────────────────┐
│  ☐ 🧤 健身手套          │
│  存放：玄关鞋柜第三层     │
└──────────────────────────┘
```
- 最左侧：复选框
- 中间：图标 + 名称 + 存放位置（小字）
- 点击行进入详情页

### 搜索框
- 顶部固定
- 圆角输入框 + 搜索图标
- 输入即搜索，无需按回车

### 弹窗（成功出门）
- 居中模态弹窗
- 鼓励文案 + 大号图标/Emoji
- "好的！"按钮关闭

## 页面布局

### Mac（宽屏）
- 最大内容宽度 480px，居中
- 一列卡片布局

### iPhone（竖屏）
- 全屏宽度
- 一列卡片，充分利用屏幕

---

## 交互反馈

| 操作 | 反馈 |
|------|------|
| 点击卡片 | 轻微缩放（scale 0.97） |
| 复选框勾选 | 打钩动画 + 绿色高亮 |
| 删除操作 | 确认弹窗，防误触 |
| 新建场景/物品 | 淡入动画 |
| 搜索无结果 | 友好空状态提示 |

---

## 图标策略

- 场景图标：默认使用 Emoji，后期可支持上传
- 物品图标：默认使用 Emoji，可上传照片替代
- Emoji 颜色丰富、免加载、免上传，适合小白起步
